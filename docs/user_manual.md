# Sky-Lark 用户手册

## 1. 简介

Sky-Lark 是一个直观、高效的系统仪表盘，帮助您实时监控应用状态并管理核心数据。

## 2. 访问系统

- **访问地址**：http://localhost:3000
- **浏览器推荐**：Chrome、Firefox、Edge、Safari（最新版）

## 3. 功能指南

### 3.1 首页仪表盘

首页是进入系统的默认视图，包含以下模块：

#### 顶部导航
- **项目标题**：显示 Sky-Lark 名称与简短说明。
- **导航入口**：首页 / 关于，可快速切换页面。

#### 状态概览
顶部展示三个状态卡片，实时反馈系统核心组件的健康状况：
1. **前端**：显示前端服务状态（绿色表示在线）与更新时间。
2. **后端 API**：实时检测后端 API 连接。如果断开，卡片会显示“后端已断开”，可点击 **重试** 按钮尝试重连。
3. **数据库**：显示数据库连接信息与初始化状态说明。

#### 数据演示
页面下方是一个交互式数据表格：
- **字段展示**：`ID / 名称 / 描述` 三列。
- **刷新数据**：点击右上角的 **刷新数据** 按钮，系统将重新从数据库拉取最新数据。
- **加载反馈**：拉取数据时会显示加载状态，避免白屏。
- **空状态提示**：当表格无数据时显示提示信息。

### 3.2 关于页面

点击顶部导航栏的“关于”链接进入。此处包含项目技术背景、核心功能列表以及版本信息。

## 4. 常见问题

**Q：为什么打开页面显示“Network Error”？**
A：请检查后端服务是否已启动。尝试运行 `docker compose up` 并等待 1 分钟。

**Q：数据表格为空？**
A：初始化启动时，系统会自动插入 3 条演示数据。如果表格为空，可能是数据初始化脚本未执行，请联系管理员查看后端日志。
